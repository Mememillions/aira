## Viral Detection (For AI Influencer Content)

### What Goes Viral in AI Influencer Space

| Type | Engagement Pattern | Example |
|------|-------------------|---------|
| **Money claims** | High shares/saves | "Made $10K with this prompt" |
| **Fear hooks** | High comments (debate) | "AI will replace designers" |
| **Listicles** | High likes (bookmark) | "5 AI tools for productivity" |
| **Hot takes** | High comments (controversy) | "ChatGPT is overrated" |
| **Drama** | High shares (gossip) | "This AI influencer is wrong" |

### Algorithm Adjustments for AI Niche

```python
# AI influencer content weights
VIRAL_SCORE = (
    (likes / followers) * 0.30 +      # Standard reach
    (comments / likes) * 0.35 +       # HIGHER weight (AI debates = viral)
    (shares / likes) * 0.25 +         # Shares (money content)
    (saves / likes) * 0.10            # Saves (tutorials)
) * 100
```

**Why higher comment weight:** AI influencers trigger debates ("Will AI replace X?"), which drives viral loops.
